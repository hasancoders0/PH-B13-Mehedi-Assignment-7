# KeenKeeper

KeenKeeper is a friendship tracking web app that helps users stay connected with important people in their life.

## Technologies Used
- React.js
- Tailwind CSS
- Recharts
- React Icons
- React Toastify

## Features
- View and manage friends
- Quick check-in with Call, Text, and Video interactions
- Timeline with filter, sort, and search
- Friendship analytics with pie chart
- Responsive design for mobile, tablet, and desktop
